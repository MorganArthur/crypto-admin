# 快速启动指南 - 策略管理和合约回测功能

## 前置条件

确保您已经安装了以下依赖：
- Python 3.8+
- Node.js 16+
- npm 或 yarn

## 安装依赖

### 后端依赖
```bash
cd backend
pip install -r requirements.txt
```

### 前端依赖
```bash
cd frontend
npm install
```

## 启动服务

### 方式一：使用启动脚本（推荐）

在项目根目录执行：
```bash
# Windows
start-full.bat
```

### 方式二：手动启动

#### 1. 启动后端服务
```bash
cd backend
python api_server.py
```

后端服务将在 http://localhost:8000 启动

#### 2. 启动前端服务（新终端）
```bash
cd frontend
npm run dev
```

前端服务将在 http://localhost:5173 启动（或其他端口，查看终端输出）

## 访问应用

打开浏览器访问前端地址（通常是 http://localhost:5173）

## 新功能使用

### 1. 策略管理

1. 点击左侧菜单 **"策略管理"**（齿轮图标）
2. 查看现有策略列表
3. 点击 **"新建策略"** 创建自定义策略
4. 填写策略信息：
   - 策略名称（唯一标识）
   - 策略描述
   - 策略类型（现货/合约）
   - 参数配置
5. 可以编辑或删除已有策略

### 2. 合约回测

1. 点击左侧菜单 **"合约回测"**（交换图标）
2. 配置回测参数：
   - **交易对**: 选择要回测的交易对（如 BTC/USDT）
   - **K线周期**: 选择时间周期（1h/4h/1d）
   - **策略**: 选择合约策略（只显示合约类型策略）
   - **时间范围**: 设置回测起止日期
   - **初始资金**: 设置初始资金（USDT）
   - **杠杆倍数**: 设置杠杆（1-125倍）
   - **策略参数**: 调整策略的具体参数
3. 点击 **"运行回测"** 按钮
4. 等待回测完成，查看结果：
   - 统计指标（收益率、最大回撤、胜率等）
   - 权益曲线图
   - 交易记录列表

## 内置策略说明

### 现货策略

#### sma_cross (双均线交叉策略)
- **原理**: 短期均线上穿长期均线买入，下穿卖出
- **参数**:
  - short_period: 短期均线周期（默认10）
  - long_period: 长期均线周期（默认30）

#### rsi (RSI相对强弱指标策略)
- **原理**: RSI低于超卖线买入，高于超买线卖出
- **参数**:
  - period: RSI周期（默认14）
  - oversold: 超卖线（默认30）
  - overbought: 超买线（默认70）

### 合约策略

#### futures_sma_cross (合约双均线交叉策略)
- **原理**: 金叉做多，死叉做空
- **参数**:
  - short_period: 短期均线周期（默认10）
  - long_period: 长期均线周期（默认30）
- **特点**: 支持双向交易，可做多和做空

## 常见问题

### Q: 为什么看不到某些策略？
A: 合约回测页面只显示合约类型的策略（type="futures"或名称包含"futures"）

### Q: 回测失败，提示数据文件不存在？
A: 确保 `backend/data/` 目录下有对应的CSV数据文件，文件名格式为 `{SYMBOL}_ohlcv_{TIMEFRAME}.csv`

### Q: 如何获取更多数据？
A: 使用"定时任务"功能自动同步数据，或在"数据查看"页面手动触发数据爬取

### Q: 自定义策略重启后消失了？
A: 当前版本自定义策略存储在内存中，重启后会丢失。后续版本会添加持久化存储。

### Q: 杠杆设置多少合适？
A: 建议从低杠杆开始测试（如5-10倍），熟悉后再逐步调整。高杠杆风险较大。
